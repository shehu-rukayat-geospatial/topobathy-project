# topobathy-project
# **Integrated UAV–Bathymetric Topobathymetric Modelling and Reservoir Capacity Assessment of Unilorin Dam, Ilorin, Nigeria**
## Project Overview
-This repository presents the methodological framework, data processing workflow, and analytical outputs of an integrated topobathymetric survey conducted at University of Ilorin Dam.
-The study integrates high-resolution UAV photogrammetry and single-beam echo sounder bathymetry within a GIS environment to generate a seamless topobathymetric Digital Elevation Model (DEM) for:
-Reservoir storage capacity estimation
-Elevation–area–capacity curve derivation
-Sedimentation assessment
-Spatial analysis of sediment deposition patterns
-The approach demonstrates the application of modern geospatial technologies for improved reservoir monitoring and water resource management.
## Study Area Characteristics
-The University of Ilorin Dam is an earth-fill reservoir commissioned in 2007 with:
-Crest length: 191 m
-Height: 8.2 m
-Original storage capacity: 1.800 × 10⁶ m³
-The reservoir functions primarily for water supply and institutional utility services.
## Research Aim
-To assess reservoir capacity loss and sedimentation using an integrated UAV photogrammetry and single-beam echo sounder bathymetric survey approach.
## Specific Objectives
-Acquire high-resolution topographic data of shoreline and surrounding terrain using UAV photogrammetry
-Collect detailed bathymetric depth measurements using GNSS-integrated acoustic sounding
-Develop an integrated topo-bathymetric surface model
-Derive updated elevation–area–capacity relationships
-Quantify sedimentation-induced storage loss
-Analyze spatial sediment thickness distribution
